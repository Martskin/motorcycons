Modifier Classes

Add the mc-spin class to an element to add a spinning animation to an icon.

<span class="mc-sprocket-rear mc-spin"></span>
<span class="mc-tire-street mc-spin"></span>
<span class="mc-tire-offroad mc-spin"></span>
<span class="mc-tire-atv mc-spin"></span>

Add the mc-rumble class to an element to add a rumbling animation to an icon.

<span class="mc-motorcycle-cruiser mc-rumble"></span>
<span class="mc-atv-farm mc-rumble"></span>
<span class="mc-engine-v-twin mc-rumble"></span>
<span class="mc-piston mc-rumble"></span>
