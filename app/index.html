Batch

A simple, responsive flexbox grid system.

Basic Usage

                <div class="batch-u-1-3">

This cell will be 33% width on all screens sizes.

                <div class="batch-u-2-3">

This cell will be 66% width on all screens sizes.

                <div class="batch-u-1-4">

This cell will be 25% width on all screens sizes.

                <div class="batch-u-1-2">

This cell will be 50% width on all screens sizes.

                <div class="batch-u-1-4">

This cell will be 25% width on all screens sizes.

                <div class="batch-u-3-8">

This cell will be 37.5% width on all screens sizes.

                <div class="batch-u-5-8">

This cell will be 62.5% width on all screens sizes.

                <div class="
                batch-u-1-8
                ">

This cell will be 12.5% width on all screens sizes.

                <div class="batch-u-7-8">

This cell will be 87.5% width on all screens sizes.

                <div class="batch-u-1-1">

This cell will be 100% width on all screens sizes.
